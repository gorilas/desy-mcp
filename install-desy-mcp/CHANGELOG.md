# Changelog - install-desy-mcp

## v1.0.0 - enero 2025

### Nuevas funcionalidades
- Servidor MCP de DESY - Sistema de Diseño del Gobierno de Aragón
- Herramientas para obtener codigo de componentes (HTML, Nunjucks, Angular)
- Herramienta para obtener propiedades de componentes
- Busqueda de componentes por nombre o descripcion
- Obtencion de guias de estilo y documentacion
- Cache de 24 horas para llms.txt con stale-while-revalidate

### Cambios de configuracion
- Configuracion hardcodeada para DESY (eliminado entrada dinamica de usuario)
- Eliminadas referencias a Twitter/X
- Anadida atribucion al proyecto original install-this-mcp
